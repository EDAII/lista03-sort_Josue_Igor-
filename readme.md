## Lista 3 Algoritmos de Ordenação com complexidade: n Log(n)

### Igor Gabriel - Josué Nascimento

Nessa lista 03 nós resolvemos implementar, no jupyter notebook, 3 Algoritmos de ordenação, sendo 2 deles O(n log n) - Merge sort e Shell sort - e um O(n+k) - Bucket sort. E com estes algoritimos geramos graficos escalares que mostram o tempo de execução levado por eles de acordo com o tamanho dos vetores(100, 1000, 10000, 100000). Esses tempos de execuções são obitidos de duas formas:
 - Valores gerado aleatóriamente;
 - Valores já ordenados;

Além disso implementamos também a geração de gráficos dinamicamente, onde é possivel ver como determinado algoritimo se comporta durante a ordenação.

Para rodá-lo basta rodar o seguinte arquivo com o python no terminal:

```
  python3 graphic.py
```

Depois é só seguir as instruções no terminal.
